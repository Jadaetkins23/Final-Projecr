Help

Prompts: When you pick a mood on the prompt page it will generate a series of words that you may use for inspiration.

Poems: The poem page generates a random poem everytime you reload the page. In addition there is also a button on the page that when clicked will generate a poem insted of reloading the page(however the website must bE used on chrome with the extention allow control allow origin in order for it to work).
